🎓 Interactive Campus Map
Visual Navigation • Smart Guidance • AI Assistant

An advanced, AI-powered campus navigation system that provides seamless visual exploration, intelligent assistance, and multilingual accessibility. This full-stack application combines modern frontend engineering with AI-integrated backend services to deliver a production-ready digital campus experience.

🌟 Overview

Interactive Campus Map enables students, visitors, and faculty to:

Navigate campus visually through an interactive map

Get turn-by-turn directions with floor-level details

Interact with an AI-powered chat assistant

Access intelligent FAQs

Explore labs, facilities, and departments

Search locations with fuzzy matching

Switch between multiple Indian languages

This project demonstrates scalable architecture, modern UI design, and real-world AI integration.
